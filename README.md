# شمارش افراد در تصویر

در این پروژه تعداد افرادی که در تصویر حضور دارند محاسبه می‌شود.

## نحوه استفاده

برای استفاده از این پروژه مراحل زیر را دنبال کنید:

1. ابتدا فایل‌های پروژه را با استفاده از دستور زیر دانلود (کلون) کنید:

   ```bash
   git clone https://github.com/sajjadshamsnia/people_count.git
   ```

2. وارد پوشه `notebooks` شوید:

   ```bash
   cd notebooks
   ```

3. فایل `people_count2.ipynb` را اجرا کنید.

## پیش‌نیازها

- Python
- Jupyter Notebook یا JupyterLab

کتابخانه‌های مورد نیاز پروژه (در صورت وجود فایل `requirements.txt`، با دستور زیر نصب کنید:)

  ```bash
  pip install -r requirements.txt
  ```
